## 关键发现摘要

### AI/ML Harness Engineering（最热门）
- OpenAI 用 Harness Engineering 交付了100万行代码零手写的产品
- Anthropic 的 V2 Harness（Opus 4.6）在3.8小时/$125完成数字音频工作站
- LangChain Deep Agents 仅通过 Harness 优化就从52.8%提升到66.5%
- Google DeepMind AutoHarness：小模型+合成 Harness 超越大模型
- SWE-agent 证明 Action Space 设计对性能的影响有时大于模型选择

### 核心实践原则
1. **改进 Harness 的 ROI 通常高于换更大的模型**
2. **最小充分工具集**（5-13个）优于大量工具（100+）
3. **上下文重置**优于上下文压缩
4. **单 Agent + 好 Harness** 通常优于复杂多 Agent 系统
5. **Eval 驱动开发**是 Harness Engineering 的最高形式
